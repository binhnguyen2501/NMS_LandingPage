NEAR Message Service
The first application
for exchanging messages on NEAR Protocol.
About
NMS is convenient service that makes it simple, easy, and secure for NEAR accounts to exchange messages with each other.
The system is completely decentralized and takes advantage of the advantages and power of the NEAR Protocol.
    Why you should use NMS?
  • You do not need to create complicated accounts, do not have to enter personal information. Simply use the NEAR account.
  • The messages is completely decentralized on the NEAR Blockchain. You don't worry about messages being controlled by 3rd parties.
  • Exchange private messages securely. The message (In Private Message mode) are completely secure, only the sender and recipient can read the content of the message. In addition, no one, any third parties, can not read the content of your messages.

Features
Message exchange
Support all basic functions for messages as: Compose new message, Reply and Forward messages. And support the message content in HTML format.
Decentralized
The NMS is completely decentralized on the NEAR Blockchain. All messages are stored on NEAR Blockchain and IPFS. So you don't worry about your messages being controlled by 3rd parties.
Privacy
The message is completely secured and privacy in "Private Message" mode. Only the sender and recipient can read the content of the message. Besides, no one, not even a third party, can not know the content of your messages.
Interface
User-friendly interface that supports 2 themes: Dark theme and Light theme
